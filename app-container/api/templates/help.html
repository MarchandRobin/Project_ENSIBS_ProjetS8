Partie 1: Obfuscation

  1. Cliquez sur le boutton "Parcourir" et sélectionner le fichier que vous voulez uploader.
  2. Cliquez sur le boutton "Importer" pour envoyer le fichier.
  3. Ajoutez des rôles au fur et à mesure que vous les créez.
  4. Sélectionnez les parties que vous voulez censurer sur le fichier importé.
  5. Cliquez sur le boutton "Télécharger" pour télécharger le fichier obfusqué.
  6. Pensez à bien sauvegarder les passphrases car vous n'y aurez plus accès

Partie 2: DésObfuscation

  1. Cliquez sur le boutton "Parcourir" et sélectionner le fichier que vous voulez uploader.
  2. Cliquez sur le boutton "Importer" pour envoyer le fichier.
  3. Rentrez votre passphrase.
  4. Cliquez sur le boutton "Créer le document" pour retrouver le document déobfusqué.
  5. Cliquez sur le boutton "Télécharger" pour retrouver le document déobfusqué.
